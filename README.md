# ML-IDS
머신러닝 기반 네트워크 패킷 탐지 및 악성코드 분류 시스템 (CICIDS2017 데이터셋 활용)

## 📊 프로젝트 개요
- 네트워크 트래픽(패킷) 데이터를 분석하여 정상/악성 유무를 분류
- 악성 패킷일 경우, 어떤 유형의 사이버 공격(Malware Type)인지 ML 모델을 통해 다중 분류(Multi-class Classification) 수행
